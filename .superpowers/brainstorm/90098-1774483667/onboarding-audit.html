Onboarding Questions — Language Audit

Current questions measured against the NLP Language Framework

Issues Found

1. Too Many Options (Q6–Q12)

Framework says: max 2 options, 3 absolute max. Currently Q6-Q12 all have 4 options + "Write my own" = 5 choices. This creates decision paralysis in an emotional context.

Example (Q8 — Abandonment Response):
❌ 4 options + write my own = 5 total
"I'd start wondering..." / "I'd figure they're busy..." / "I'd enjoy the space" / "I'd notice it, but try not to..."

2. Q10 (Childhood Safety) Feels Like a Therapy Intake

"Growing up — was home a place that felt mostly safe and steady for you?" is too direct for a first session. The framework says: never sound clinical, never sound like a therapy intake form.

Also has 4 options ranging from "Mostly yes" to "Not really — it was hard" — feels like a trauma screening scale.

3. Options Are Transparent Labels

Users can easily see they're being profiled. "I'd start wondering if something's wrong" = anxious, "I'd enjoy the space" = avoidant. The framework says assessment should be invisible — woven in, not presented as a test.

4. Bridges Are Repetitive

Q2 uses "Good to know. Let's keep going." for all 8 options. Q14 uses "Got it." for all 3. Bridges should feel specific and affirming — not templated.

5. Q14 Does Double Duty Awkwardly

"What is it you're really hoping for — the thing you can't quite put words to yet" asks for a growth goal AND check-in frequency in the same question. The growth prompt may freeze users who aren't ready to articulate something deep.

6. No Presupposition in Assessment Questions

Q6–Q9 ask "what happens when..." — these are descriptive, not forward-moving. The framework says: presuppose forward movement, offer choices of how, not whether. These questions profile but don't pull the user forward.

What's Working Well
  • Q1-Q5 flow naturally — warm, conversational, good pacing
  • Peter's voice in Q1 ("I'm going to be with you every step of the way") is strong
  • Dynamic partner name insertion works well throughout
  • Q6's framing ("a short reply, a change in tone, a quiet night") is evocative
  • Several bridges are excellent: "That takes courage. Leading the way says a lot about you."
