Implementation note

Each question in questions.ts includes a bridges map: { [optionKey]: string }. When the user selects an option, the bridge message is shown as a Peter bubble for ~1.5s before the next question animates in. For "write my own" answers, a generic warm acknowledgment is used. Zero API calls — all pre-written copy.
