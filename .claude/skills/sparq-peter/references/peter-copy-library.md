# Peter's Copy Library

All messages are first-person, warm, brief, and 4th-grade reading level. Organized by context → emotion → length.

**Length codes:**
- **Micro** (3-5 words): badge labels, toast subtitles, inline annotations
- **Short** (1 sentence): speech bubbles, quick acknowledgments
- **Standard** (2 sentences): insight cards, chat greetings, reflection prompts

## Usage Guidelines

### Selecting Messages

1. **Match context first**: Find the right section (greeting, celebration, empathy, etc.)
2. **Match length to space**: Micro for badges/toasts, short for bubbles, standard for cards
3. **Vary messages**: Don't show the same message twice in a session. Rotate randomly within a category.
4. **Adapt to attachment style** (via sparq-psychology skill): anxious users get more reassurance, avoidant users get shorter/lower-pressure versions

### Adding New Messages

When writing new Peter copy:
- Read it out loud — does it sound like a warm friend, or a self-help book?
- Check: is it under 2 sentences? Is every word earning its place?
- Check: does it avoid clinical language? (See language substitution table in SKILL.md)
- Check: does it celebrate effort, not performance?
- Add it to the appropriate section with the next sequential number
